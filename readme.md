# Өгөгдлийн сангийн удирдлагын бичиг баримт болон нуу үгийн бодлогыг хэрэгжүүлэх

##  Асуулт 9. 
Plain text байдлаар нууц үг хадгалсан тохиолдолд үүсэх аюулгүй байдлын эрсдэлийг кейс жишээгээр тайлбарлана уу.

## Хариулт 
Plain text байдлаар нууц үг хадгалах гэдэг нь хэрэглэгчийн нууц үгийг ямар ч шифрлэл, hash, salt зэрэг хамгаалалтгүйгээр яг бичсэнээр нь өгөгдлийн санд хадгалахыг хэлнэ. Энэ нь МАБ-ын хамгийн том алдаа гэж тооцогддог. Учир нь систем халдлагад өртсөн эсвэл дотоод ажилтан мэдээлэлд хандсан тохиолдолд бүх нууц үг шууд харагддаг.

## Дүгнэлт 
Plain text password хадгалах нь системийн хамгийн ноцтой аюулгүй байдлын алдаа юм. Хэрэв системд халдлага орвол бүх хэрэглэгчийн нууц үг ил болж, аккаунт хакердах, хувийн мэдээлэл алдагдах, санхүүгийн эрсдэлтэй. Иймээс системүүд password-ийг hash болон salt ашиглан хамгаалж хадгалах ёстой.
